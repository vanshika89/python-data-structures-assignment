# E-Commerce Control Flow Utilities (Assignment 2)

A collection of Python scripts focused on order processing logic using control flow constructs: `if/elif/else`, `for` loops, and `while` loops with `break/continue`.

## Project Structure
* `Task1_DiscountRules.py`: Input validation and tiered discount logic.
* `Task2_MultipleOrders.py`: Batch processing of orders with revenue summation.
* `Task3_UserMenu.py`: Interactive menu-driven interface for order management.
* `Task5_LoopControl.py`: Data cleaning and error handling using break/continue.
* `main.py`: A master script to execute all tasks sequentially.

## How to Run
1. Ensure you have **Python 3.10+** installed.
2. Open your terminal in the project directory.
3. To run the full suite:
   ```bash
   python main.py
   
4. To run an individual task:
   ```bash
   python Task1_DiscountRules.py
   python Task2_MultipleOrders.py
   python Task3_UserMenu.py
   python Task5_LoopControl.py
