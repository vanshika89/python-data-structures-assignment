# E-Commerce Inventory Utility (Assignment 1)

A Python-based utility to manage product inventory using core data structures: Lists, Tuples, Sets, and Dictionaries.

## 📁 Project Structure
* `Task1_ProductCollections.py`: Basic list and tuple operations (Indexing, Slicing, and Mutability).
* `Task2_Categories.py`: Set operations for unique categories and duplicate handling.
* `Task3_ProductPricing.py`: Dictionary-based pricing, average calculation, and min/max lookups.
* `Task4_CombinedOperations.py`: Complex data mapping and Lambda-based queries.
* `main.py`: The entry point that executes the entire assignment suite.

## 🚀 How to Run

3. Open your terminal or command prompt in that folder.
